### 要求PHP版本>=7.4
### 目前功能  
- [x] 随机图片（这不是废话吗
- [x] 自动识别桌面端与移动端
- [x] 方便化添加图片链接
- [x] 轻量化（雾
- [ ] 更加准确的识别
- [x] 本地化（服务器带宽够大就是任性
- [x] 多文件夹自动抓取（老懒狗了，懒得重命名

### 添加图片的方法
将整个图片文件夹扔进img目录下即可  
桌面端图片文件夹命名规则：xxxx_ppic  
移动端图片文件夹命名规则：xxxx_mpic  
xxx可依自己喜好随意更改，可以是时间，可以是分类名称
