## 本api基于PHP  
### 要求PHP>=7.4
### 目前功能  
- [x] 随机图片（这不是废话吗
- [x] 自动识别桌面端与移动端
- [x] 方便化添加图片链接
- [x] 轻量化（雾
- [ ] 更加准确的识别

### 添加图片的方法
只需要将图片链接分别添加进image-list-desktop或者image-list-mobile  
image-list-desktop及对应桌面端将显示的图片  
image-list-mobile及对应移动端将显示的图片
